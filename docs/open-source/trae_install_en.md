
# **Trae Installation Guide**

## Trae Installation

1. Visit the [Trae website](https://www.trae.com.cn) and download the installer for your operating system.
2. After installation, sign up or log in to your Trae account.
3. Configure agent preferences such as language and plugin permissions as needed.
4. Test the agent by entering a command in the dialog, e.g., `Hello`.

## OneSkills Installation

1. Download the OneScience VSCode Plugin `.vsix` [package](https://gitee.com/onescience-ai/onescience-vscode-plugin/releases/download/v0.1.4/onescience-vscode-plugin-0.1.4.vsix).
2. Open Trae, go to the **Extensions** panel, click `...` in the top-right corner, and select **Install from VSIX**.
3. Select the `.vsix` file and wait for the installation to complete.
4. After installation, click the Extensions icon on the left sidebar, select **Workspace**, then click **Create Agent** — an agent with OneSkills will be automatically created.
5. Select the created agent from the dialog to start interacting.
