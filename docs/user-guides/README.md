# User Guides

本目录面向 `OneSkills` 的普通用户、贡献者和领域专家。

## 当前指南

- `extend_domain_experience.md`：如何把你的科研领域经验接入 `OneSkills`
- `claude_codex_plugin_install.md`：Claude Code 与 Codex 中安装 OneSkills skills 和可选 SCnet MCP 的方法

## 使用建议

- 如果你想补充新的科研领域知识，先读 `extend_domain_experience.md`
- 如果你想把 OneSkills 安装到 Claude Code 或 Codex，读 `claude_codex_plugin_install.md`
- 如果你只是想使用现有 skills，优先从根目录 `README.md` 开始
