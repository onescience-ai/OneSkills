# 贡献指南

感谢你对 `OneSkills` 的贡献。


## 贡献范围

公开贡献应聚焦于以下目录：

- `skills/`
- `docs/`

请勿将私有维护材料添加至公开产品界面中。

## 变更前须知

请先阅读相关公开文档：

- [README.md](README.md)

保持变更最小化：

- 优先扩展现有技能，而非新增技能

## 文档要求

当行为发生变更时，需同步更新文档。

常见更新位置：

- 产品概述：`README.md`
- 公开扩展指南：`docs/`

## Pull Request 规范

一份有效的 PR 描述应包含以下信息：

- 变更内容
- 哪些面向用户的行为发生了变化
- 涉及哪些目录
- 执行了哪些公开检查
- 是否存在兼容性注意事项
