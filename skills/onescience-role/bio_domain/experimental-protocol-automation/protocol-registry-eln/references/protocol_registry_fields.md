# 协议与注册字段

## Protocol

Title、version、purpose、source、operator、date、materials、equipment、steps、expected outputs、deviations。

## Sample

Sample id、organism、tissue/cell type、condition、donor/patient id、collection date、storage、consent/deidentification status。

## Construct

Construct id、sequence/map、backbone、insert、promoter、marker、host、verification evidence、storage location。

## Inventory

Container、box、well/tube position、volume、concentration、lot、expiration 和 freeze-thaw count。
