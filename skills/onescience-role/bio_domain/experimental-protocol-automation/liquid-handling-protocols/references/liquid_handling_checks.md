# 液体处理检查

## 体积和耗材

- 每个 transfer 必须在 pipette min/max 体积范围内。
- Reservoir 和 tube 要考虑 dead volume。
- Well capacity、mix volume 和 blowout 位置必须安全。

## 实验设计

- 记录 positive/negative/no-template control。
- Plate map 应包含 sample id、condition、replicate 和 reagent。
- PCR/qPCR setup 需考虑冷却模块或短时间上机。

## 上机前

必须运行 simulation，检查 tip 数量、碰撞风险、模块位置、液体总量和人工装载步骤。
