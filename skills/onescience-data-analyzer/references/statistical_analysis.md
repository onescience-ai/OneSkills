# 统计分析工作流

## 工作流目标

对数据执行统计分析，生成统计指标与分析结果。

## 执行步骤

### 1. 数据加载与验证
- 读取数据文件
- 检查数据完整性
- 识别数据类型与维度

### 2. 描述性统计
- 计算基本统计量（均值、中位数、标准差、方差）
- 识别异常值
- 生成数据分布特征

### 3. 推断统计
根据需求执行：
- 相关性分析（Pearson/Spearman）
- 假设检验（t-test, ANOVA, Chi-square）
- 回归分析（线性/非线性）

### 4. 结果整理
- 生成统计表格
- 标记显著性结果
- 输出结构化数据

## 输出内容

```json
{
  "descriptive_stats": {...},
  "correlation_matrix": [...],
  "test_results": {...},
  "summary_table": "path/to/stats_table.csv"
}
```
